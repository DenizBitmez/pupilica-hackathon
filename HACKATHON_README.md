# 🚀 Tarih-i Sima - Hackathon Projesi

## 🎯 Proje Özeti

**Tarih-i Sima**, yapay zeka destekli tarihi karakterlerle etkileşimli eğitim platformudur. Kullanıcılar tarihi figürlerle gerçek zamanlı konuşabilir, onların hikayelerini dinleyebilir ve kapsamlı eğitim araçları ile tarihi olayları keşfedebilir.

## ✨ Hackathon Özellikleri

### 🤖 AI-Powered Character Simulation
- **Ollama AI** entegrasyonu ile gerçek zamanlı konuşma
- **3 Tarihi Figür**: Fatih Sultan Mehmet, Atatürk, Napoleon
- Karakteristik kişilik modelleme
- Tarihi doğruluk garantisi
- Türkçe dil desteği

### 🎭 Game-Quality Avatar System
- **CSS3 3D Transform** ile oyun kalitesinde avatarlar
- Dinamik mimikler ve animasyonlar
- Konuşma sırasında gerçekçi dudak hareketleri
- Karakteristik aksesuarlar (taç, şapka, askeri başlık)
- Göz kırpma ve konuşma efektleri

### 🗣️ Advanced Voice Synthesis
- **Web Speech API** entegrasyonu
- Karakteristik ses tonları (Fatih: yavaş/derin, Atatürk: hızlı/karizmatik, Napoleon: aksanlı)
- Gerçek zamanlı lip-sync
- TTS ayarları (hız, perde, ses seviyesi)
- Dinamik ses seviyesi simülasyonu

### 🌍 Interactive Historical Maps
- **Leaflet.js** ile interaktif haritalar
- 50+ tarihi olayın konumları
- Zaman çizelgesi entegrasyonu
- Görsel tarih keşfi
- Sebep-sonuç ilişkileri

### 🏆 Advanced Gamification System
- **7 Seviye Sistemi**: Başlangıç Öğrencisi → Tarih Profesörü
- **XP ve Para Sistemi**: Coin (💰) ve Gem (💎)
- **12 Başarım**: 5 farklı kategori ve nadirlik seviyeleri
- **5 Rozet**: Özel başarı rozetleri
- **Günlük Seri**: Düzenli çalışma takibi
- **Görev Sistemi**: Günlük ve haftalık meydan okumalar

### 🔐 User Authentication & Personalization
- **Güvenli Giriş Sistemi**: Email/şifre ile giriş
- **Demo Hesaplar**: Hızlı test için hazır hesaplar
- **Kişisel Profil**: Detaylı kullanıcı bilgileri
- **Veri Güvenliği**: LocalStorage ile güvenli veri saklama
- **İlerleme Takibi**: Kişisel öğrenme istatistikleri

### 📚 Comprehensive Educational Tools
- **İnteraktif Not Alma**: Kategorize edilmiş, etiketli not sistemi
- **Olay Bağlantı Haritası**: Tarihi olaylar arası ilişkiler
- **Karşılaştırmalı Analiz**: Farklı tarihi figürleri karşılaştırma
- **Quiz Sistemi**: 8 soru ile kapsamlı test
- **Öğrenme İpuçları**: Etkili öğrenme teknikleri

### 🎨 Modern UI/UX
- **Tailwind CSS** ile responsive tasarım
- **Glassmorphism** efektleri
- **Smooth animasyonlar**
- **Modal sistemleri**
- **Progressive Web App** özellikleri

## 🛠️ Teknoloji Stack

### Frontend
- **React 18** - Modern UI framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first CSS
- **Socket.io-client** - Real-time communication
- **Leaflet** - Interactive maps
- **Web Speech API** - Voice synthesis
- **Shadcn/ui** - Modern UI components
- **Heroicons** - Icon system

### Backend
- **Flask** - Python web framework
- **Socket.io** - Real-time communication
- **Ollama** - Local AI model
- **gTTS** - Text-to-speech
- **Flask-CORS** - Cross-origin support

### Authentication & Data
- **LocalStorage** - Client-side data persistence
- **React Context** - State management
- **TypeScript Interfaces** - Type safety

## 🚀 Kurulum

### Gereksinimler
- Node.js 18+
- Python 3.8+
- Ollama (AI model)

### Backend Kurulumu
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend Kurulumu
```bash
cd frontend
npm install
npm run dev
```

## 🎮 Kullanım

### 1. Temel Kullanım (Giriş Yapmadan)
1. **Karakter Seçimi**: Tarihi figürü seçin
2. **Avatar Etkileşimi**: Karakterin hikayesini dinleyin
3. **Temel Özellikler**: Quiz, Timeline, Map, Öğrenme İpuçları

### 2. Gelişmiş Kullanım (Giriş Yaparak)
1. **Giriş Yap**: Demo hesap kullanın veya yeni hesap oluşturun
