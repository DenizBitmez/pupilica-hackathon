# 🏛️ Tarih-i Sima: Yapay Zeka Destekli Tarih Eğitimi Platformu

## 📖 Proje Hakkında

Tarih-i Sima, tarihi öğrenme deneyimini interaktif ve kişisel bir seviyeye taşıyan modern bir eğitim platformudur. Öğrenciler ve tarih meraklıları, tarihi figürlerle doğrudan sohbet edebilir, tarihi olayları görselleştirebilir ve kişisel ilerlemelerini takip edebilir.

## ✨ Ana Özellikler

### 🤖 Yapay Zeka Destekli Tarih Eğitimi
- **3 Tarihi Figür**: Fatih Sultan Mehmet, Mustafa Kemal Atatürk, Napoleon Bonaparte
- **Gerçekçi Avatar Animasyonları**: Konuşma, dudak hareketi, göz kırpma efektleri
- **Karakter Özel Sesler**: Her tarihi figür için özelleştirilmiş ses tonları
- **Text-to-Speech**: Türkçe seslendirme sistemi

### 🎮 Oyunlaştırma Sistemi
- **Seviye Sistemi**: 7 farklı seviye (Başlangıç Öğrencisi → Tarih Profesörü)
- **XP ve Para Sistemi**: Coin (💰) ve Gem (💎) kazanma
- **Günlük Seri**: Düzenli çalışma takibi
- **Başarımlar**: 5 farklı kategori ve nadirlik seviyeleri
- **Rozetler**: Özel başarı rozetleri

### 📚 Eğitim Araçları

#### Temel Özellikler (Giriş Yapmadan)
- ✅ **Quiz Sistemi**: 8 soru ile bilgi testi
- ✅ **Zaman Çizelgesi**: Kronolojik tarih görselleştirmesi
- ✅ **Tarihi Harita**: Önemli konumları keşfetme
- ✅ **Öğrenme İpuçları**: Etkili öğrenme teknikleri

#### Gelişmiş Özellikler (Giriş Gerekli)
- 🔒 **İnteraktif Not Alma**: Kategorize edilmiş, etiketli not sistemi
- 🔒 **Olay Bağlantı Haritası**: Tarihi olaylar arası ilişkiler
- 🔒 **Karşılaştırmalı Analiz**: Farklı tarihi figürleri karşılaştırma
- 🔒 **İlerleme Takibi**: Detaylı öğrenme istatistikleri
- 🔒 **Oyunlaştırma**: Tam özellikli oyunlaştırma sistemi

### 🔐 Kullanıcı Sistemi
- **Güvenli Giriş**: Email/şifre ile giriş
- **Demo Hesaplar**: Hızlı test için hazır hesaplar
- **Kişisel Profil**: Detaylı kullanıcı bilgileri
- **Veri Güvenliği**: LocalStorage ile güvenli veri saklama

## 🛠️ Teknolojiler

### Frontend
- **React.js 18** + **TypeScript**
- **Tailwind CSS** - Modern UI tasarımı
- **Heroicons** - İkon sistemi
- **Lucide React** - Ek ikonlar
- **React Router** - Sayfa yönlendirme
- **React Query** - Veri yönetimi

### Backend
- **Python Flask** - API sunucusu
- **WebSocket** - Gerçek zamanlı iletişim
- **Text-to-Speech** - Seslendirme sistemi

### UI Components
- **Shadcn/ui** - Modern UI bileşenleri
- **Custom Animations** - Özel CSS animasyonları
- **Responsive Design** - Mobil uyumlu tasarım

## 🚀 Kurulum

### Gereksinimler
- Node.js 18+
- Python 3.8+
- npm veya yarn

### Frontend Kurulumu
```bash
cd frontend
npm install
npm run dev
```

### Backend Kurulumu
```bash
cd backend
pip install -r requirements.txt
python app.py
```

## 📱 Kullanım Kılavuzu

### 1. İlk Kullanım
1. Uygulamayı açın
2. Bir tarihi figür seçin (Fatih, Atatürk, Napoleon)
3. "Hikayeyi Dinle" ile başlayın
4. Temel özellikleri keşfedin

### 2. Giriş Yapma
1. "Giriş Yap" butonuna tıklayın
2. Demo hesaplardan birini seçin VEYA
3. Yeni hesap oluşturun
4. Gelişmiş özellikler aktif olur

### 3. Gelişmiş Özellikler
- **📝 Not Alma**: İnteraktif not sistemi
- **🗺️ Olay Haritası**: Bağlantıları keşfet
- **⚖️ Karşılaştırma**: Analiz yap
- **📊 İlerleme**: Takip et
- **🎮 Oyunlaştırma**: Kazanç sistemi
